Plots

Add / Change Plot

A plot log is required before adding/changing a plot.

RSS and HTC plots are already in the system and only need to be edited. During ESS it may be necessary to add a plot. Plots may only be added in an ESS community.

Once a plot has been added the edit plot button should appear showing the plot identifier. If the button is disabled, check the plot log history. If a current plot log does not exist, you will need to add one before the plot may be changed.

Once at the plot, you need to:

  • confirm the GPS point;
  • enter the number of households
  • estimate the number of eligible members that live on the plot
